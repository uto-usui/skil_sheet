# スキルシート

こんにちは、ウェブデベロッパーの臼井優斗(うすいゆうと)です。  
ここに私のスキルシートを記述します。




## 概要

私は28歳で奈良出身のウェブデベロッパーです。  
音楽大学を経て、25歳のころ、  
ミュージシャン兼音響エンジニアから  
キャリアチェンジをしました。

ベンチャー企業でフロントエンドのチーフを務め、  
４人から６人のチームで、  
大規模ECサイトの開発や教育機関や和服ファッションサイト、  
各種プロモーションサイトやアーティストサイトなどを経験しました。

私が得意なことは、  
HTML5.1に準拠したセマンティックなコーディングと、  
堅牢なCSS設計、フレンドリーなアニメーションです。

クライアントのビジネスやブランドにフォーカスした、  
UI/UXの提案と実装も積極的に行うエンジニアです。

コーディングの他には、  
ウェブ／グラフィックデザイン、  
コードレビュー、社内勉強会の主催、プロジェクト管理、  
人事／育成、見積もり、フォトディレクション、  
などを担当しました。

現在はthree.js／Reactなどを中心に、  
JavaScriptでのメディアアート表現や  
ウェブコンポーネントを学んでいます。

## コメント

私は、モノやヒトやコトを売ることと、  
ファンタジーが好きです。  
いつもステージに上げるようなイメージで、
クライアントのビジネスの着地点を考察し、  
素敵にライティングするスキルを磨いています。  
私のエンジニアリングで、  
魅力を自由に引き出すのに必要となる土台を用意するための、  
堅牢で美しいHTMLやCSSを生成することができます。  
そこに新しいウェブの仕様や世の中のトレンドを  
いちはやくキャッチアップして擦り合せることで、  
UI/UXをより豊かなものにします。  
また、新鮮さと没入感を演出できるような、  
JavaScriptのファンタジックなアート表現を研究しています。  

そして、  
これらをチームで実行するために努力するスキルと、仕事をたのしむ気持ち、  
こどもっぽい探究心と向上心は誰にも負けません。
